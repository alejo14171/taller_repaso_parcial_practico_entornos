# Chatbot con FastAPI y Gemini

## Descripción
Este proyecto consiste en una aplicación web que permite enviar mensajes desde un frontend en HTML, CSS y JavaScript hacia un backend desarrollado con FastAPI, el cual se conecta con la API de Gemini para generar respuestas.

## Tecnologías utilizadas
- Python
- FastAPI
- Gemini API (google.genai)
- HTML, CSS, JavaScript

## Estructura del proyecto
- main.py → Backend con FastAPI
- index.html → Frontend
- requirements.txt → Dependencias
